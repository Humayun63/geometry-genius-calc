What is DOM and what is the purpuse of DOM?

Document Object Model or DOM is a programming interface for HTML and XHTML documents. It represents the webpage as an object so that webpage can be modified through a programming language. Needless to say, the object has its own property and method. So by using that property and method of DOM we can interact with the webpage easily. Anyone can create, remove or modify elements, can add/remove/modify attributes and styles, and especially can create events through DOM.
In short, DOM is an API (application programming interface) for the website. The purpose of using DOM is to interact with the webpage easily.

How will you select HTML elements using DOM?

There are some methods for selecting HTML elements like getElementById, getElementsByClassName, getElementsByTagName, querySelector, and querySelectorAll.

getElementById() method takes an id as a parameter and returns the element of that id. Code Look like: document.getElementById('idName')

getElementByClassName() method takes a class name as a parameter and returns a HTMLCollection which is an array like object. Code looks like: document.getElementByClassName('className')

getElementByTagName() method takes a tag name as a parameter and returns a HTMLCollection which is an array like object. Code looks like: document.getElementByTagName('tagName')

querySelector() method can take tag name, id, and class as parameter and it returns the first match of the parameter. Code looks like: document.querySelector('parameter')

querySelectorAll() method can take tag name, id, and class as parameter and it returns a nodeList which is an array like object. Code looks like: document.querySelectorAll('parameter')

querySelector and querySelectorAll are most likely css selector. You can select element like you select on your css code!

Note: 'Array like object' is most likely an array. But you can't use some method like push() pop() and etc. To use all the element you can use for loop or for of loop.

What is event bubble?

Event bubble is an idea of event propagation of DOM. It says when an event is called then it propagated to its parent and calls the parent event and then that's element parent and goes on until reaches the root of the file. To stop event bubble you can use event.stopPropagation() methode.

What is a callback function and why will you use it?

When a function is passed as an argument to another function then it's called a callback function. A callback function executes inside the main function when it is called.
When you want a function to be called after the execution of a function, then you can use the first function as a callback function pretending as the second function is the main function.
